# Relativistic Plasma Mirror: High-Harmonic Generation (HHG) Analysis

A quantitative Python time-series and signal processing pipeline developed for computational physics assignment at **Imperial College London**. 

This notebook processes 1D Particle-in-Cell (PIC) simulation outputs (generated via the `EPOCH` physics solver) to model sub-cycle electromagnetic interference across high-intensity laser regimes (10^16 to 10^20 W/cm²).

## Core Quantitative Methods Displayed
* **Domain Transformation:** Application of discrete Fast Fourier Transforms (`scipy.fft`) to transition raw electric field time-series data into the frequency spectrum.
* **Instantaneous Signal Extraction:** Gaussian window filtering and Hilbert transforms (`scipy.signal.hilbert`) to isolate slowly-varying amplitude envelopes and carrier phase offsets.
* **Automated Peak Detection:** Custom local-maxima search algorithms built to account for radiation pressure-induced Doppler redshift trends.
* **Multi-Variable Parameter Mapping:** 2D log-scale parameter sweeps generated across phase-space using `Xarray` and `Matplotlib`.

## Tech Stack
* **Core:** Python 3, Jupyter Notebook
* **Libraries:** `NumPy`, `SciPy`, `Xarray`, `Matplotlib`, `IPyWidgets`
